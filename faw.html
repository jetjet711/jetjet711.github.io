Aerial Detection of Fall Armyworm (FAW) Infestation in Corn Field

This project integrates drone imagery and AI for early detection of Fall Armyworm infestation in cornfields. It uses computer vision models to analyze live aerial footage and determine the infestation percentage of crops.

Tech Stack

  • Python (YOLOv5 / OpenCV)
  • Flask Web App
  • DJI SDK + Drone Camera Feed
  • OBS → WebRTC Streaming Integration
